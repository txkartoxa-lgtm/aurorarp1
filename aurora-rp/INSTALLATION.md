# Aurora RP - DarkRP Server

**Aurora RP** - это полноценный DarkRP сервер для Garry's Mod с красивым дизайном, реалистичными профессиями и атмосферным освещением.

## 🌟 Особенности

### Основное
- ✅ **Третье лицо** - переключение камерой (клавиша V)
- ✅ **Красивое меню профессий** (F2) - Полицейский, Врач, Таксист и многие другие
- ✅ **Донат меню** (F3) - магазин с VIP статусами, оружием, транспортом
- ✅ **Система ключей и дверей** - покупка, добавление/удаление владельцев
- ✅ **Голосовой чат** -proximity-based с визуальными индикаторами
- ✅ **Атмосферное освещение** - динамическая смена времени суток

### HUD Интерфейс
- ⏰ **Время** в верхнем правом углу (с названием Aurora RP)
- ❤️ **Здоровье, Броня, Голод, Деньги** в нижнем левом углу
- 📊 **Прогресс бары** для наглядности

### Команды
- `!knock` - постучаться в дверь
- `!addowner [игрок]` - добавить владельца двери
- `!removeowner [игрок]` - удалить владельца двери
- `!job [название]` - выбрать профессию
- `!donate [id]` - запрос на покупку доната

### Настройки
- Стартовые деньги: **5000$**
- Здоровье: **100 HP**

- Голод: **100**

## 📦 Установка

1. Скачайте карту из Steam Workshop:
   - **rp_downtown_tits_v2**: https://steamcommunity.com/sharedfiles/filedetails/?id=1527403485

2. Поместите папку `aurora-rp` в директорию:
   ```
   garrysmod/addons/aurora-rp/
   ```

3. Запустите сервер с картой:
   ```
   map rp_downtown_tits_v2
   ```

4. Убедитесь что установлен **DarkRP** мод

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| F2 | Открыть меню профессий |
| F3 | Открыть донат меню |
| V | Переключить вид от третьего лица |
| E | Взаимодействие с дверьми |
| !knock | Постучаться в дверь |

## 🔧 Конфигурация

Все настройки находятся в файле:
```
lua/autorun/sh_config.lua
```

Вы можете изменить:
- Стартовые деньги
- Дистанцию голосового чата
- Настройки третьего лица
- Цены на донат
- И многое другое!

## 📝 Структура файлов

```
aurora-rp/
├── gamemode/
│   ├── init.lua              # Основной файл гейммodes
│   ├── cl_init.lua           # Клиентская инициализация
│   ├── shared.lua            # Общие функции
│   ├── cl_hud.lua            # HUD интерфейс
│   ├── cl_jobs_menu.lua      # Меню профессий
│   ├── cl_donate_menu.lua    # Донат меню
│   └── cl_thirdperson.lua    # Система третьего лица
├── lua/
│   └── autorun/
│       ├── sh_config.lua     # Конфигурация
│       ├── cl_voicechat.lua  # Голосовой чат
│       └── server/
│           ├── sv_jobs.lua   # Настройка профессий
│           └── sv_lighting.lua # Освещение
└── addon.json                # Информация об аддоне
```

## 💡 Советы

- Для смены вида камеры нажмите **V**
- Используйте **F2** для быстрой смены профессии
- Постучаться в чужую дверь можно командой **!knock**
- Владелец двери может добавить друзей через **!addowner**

## 👥 Поддержка

Для изменения донат магазина обратитесь к файлу:
```
gamemode/cl_donate_menu.lua
```

Там вы сможете добавить свои товары и цены.
