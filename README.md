# 🌌 AURORA RP — DARKRP СЕРВЕР

**Aurora RP** — это полноценный DarkRP сервер с атмосферой реализма, современным UI дизайном в стиле северного сияния (Aurora) и качественным геймплеем.

## 🎨 Дизайн-система "Aurora"

### Цветовая палитра
- **Основной фон:** #0A0A1A (глубокий тёмно-синий)
- **Вторичный фон:** #0D1B2A (тёмно-синий)
- **Акцент 1:** #6C63FF (фиолетовый)
- **Акцент 2:** #00D4FF (голубой неон)
- **Акцент 3:** #7B2FBE (тёмно-фиолетовый)
- **Свечение:** #A78BFA (светло-фиолетовый)

### Визуальные эффекты
- Анимированный градиент северного сияния
- Glassmorphism эффект на панелях
- Glow-эффект на кнопках при наведении
- Плавные переходы (0.3s ease)

## ⚙️ Установка

### Требования
1. Garry's Mod Server
2. DarkRP модификация
3. ULX + ULib (администрирование)
4. Карта rp_downtown_tits_v2

### Шаги установки

1. **Скопируйте файлы сервера:**
   ```bash
   cp -r garrysmod/* /path/to/server/garrysmod/
   ```

2. **Установите необходимые аддоны:**
   - DarkRP (GitHub)
   - DarkRPmodification (GitHub)
   - ULX + ULib (Workshop)
   - rp_downtown_tits_v2 (Workshop: 1527403485)

3. **Настройте server.cfg:**
   ```
   hostname "Aurora RP | Реалистичный DarkRP | 18+"
   sv_maxplayers 64
   ```

4. **Запустите сервер**
